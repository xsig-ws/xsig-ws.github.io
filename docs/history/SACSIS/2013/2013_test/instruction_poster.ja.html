SACSIS2013 ポスター投稿方法

SACSIS2013では，論文投稿システムとしてEasyChairを利用します．EasyChairを利用したことがある場合には，1. EasyChairのアカウント作成手順を読み飛ばして，2. SACSIS2013への論文投稿手順を御覧ください．EasyChairを利用したことがない場合には，以下の手順をご覧ください．

1. EasyChairのアカウント作成手順

  1. SACSIS2013 EasyChairサイトへアクセスしてください．
  2. "sign up for an account"をクリックしてください．
  3. "Step 1"で表示されている2つの単語を入力し，"continue"をクリックしてください．
  4. "Step 2"でFirst name, Last name, Emailアドレスを入力し，"continue"をクリックしてください．
  5. "Account Application Received"と出力され，仮手続が完了します．
  6. Step 2で登録したEmailアドレスにEasyChairシステムから"EasyChair account confirmation"という件名のメールが送られてきます．本文中のURLをクリックしてください．
  7. "Signing Up for EasyChair: Last Step"と出力された画面で，User name (EasyChairシステムにログインする際に利用する)，所属，パスワード等の情報を入力し，"create my account"をクリックしてください．
以上で，EasyChairのアカウント作成手続きが終了します．

2. SACSIS2013へのポスター投稿手順

  1. SACSIS2013 EasyChairサイト:
    https://www.easychair.org/conferences/?conf=sacsis2013
    へアクセスしてください．
  2. User name，Passwordを入力して，"Sign in"をクリックしてください．
  3. 上部メニューの"New Submission"をクリックしてください．
  4. 連絡先を入力してください．このページでは，論文が日本語で書かれている場合は日本語で，英語の場合は英語で入力してください．
  5. 著者全員の情報を入力してください．代表者には，"Corresponding author"にチェックを入れてください．なお，論文が日本語で書かれている場合は日本語で，英語の場合は英語で入力してください． 入力欄が足りない場合は，"Click here to add more authors"をクリックしてください．
  6. 論文タイトルを入力してください．論文が日本語で書かれている場合は日本語で，英語の場合は英語で入力してください．
    概要は，100文字程度の短いものでかまいません．
    カテゴリーについては，投稿する論文に最も合致するものを1つ選んで下さい．
    キーワードは，3つ以上入力してください．
    概要，カテゴリー，キーワードは，査読者の割り当てと賞審査員の割り当てに利用されます．
  7. 学生ポスターとして投稿する場合は，"Submit this poster paper as a student poster / このポスター論文を学生ポスターとして投稿する"に忘れずにチェックを入れてください．学生ポスターとして投稿する場合には，どの著者が学生かも指定してください．
  8. ポスター論文として，PDF形式のファイルをアップロードしてください．
  9. 入力内容を確認した後，"Submit"をクリックしてください．
  10. 論文アップロード〆切まで，論文のアップロード，投稿した論文の差し替えが可能です． 再度EasyChairシステムにログインし，上部メニューの"Paper <論文番号>"をクリックし，"Submit a new version"で論文のアップロード，差し替え手続きをしてください．
以上で，論文登録およびアップロードは完了します．
