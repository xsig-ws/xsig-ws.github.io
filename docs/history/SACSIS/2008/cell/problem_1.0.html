概要

たとえば、以下の3変数による連立一次方程式の解は、 消去法により求めることができます。

3x+ y+4z=17
x +5y+9z=38
2x+6y+5z=29

↓

3x+y+4z=17
(14/3)y+(23/3)z= 97/3
(16/3)y+(7/3)z= 53/3

↓

3x+y+4z=17...(1)
(14/3)y+(23/3)z=97/3...(2)
(-135/21)z=-405/21 ...(3)

以上の消去を、前進消去と呼びます。

解は、以下のように求めることができます。

  • 式(3)より、 z = 3 ...(4)
  • 式(2)に、式(4)を代入して、y = 2 ...(5)
  • 式(1)に、式(4)と式(5)を代入して、x = 1。

以上の操作を、後退代入と呼びます。

前進消去と後退代入などを利用して、与えられた連立一次方程式を解くことを、 連立一次方程式の解法と呼びます。

左辺の係数を行列表記、右辺の値と解の変数をベクトル表記します。すなわち、

(314) (x) (17)
A= 159 ,      x= y ,      b= 38
265 z 29

とし、連立一次方程式

A x = b

の解ベクトル x を求めるのが問題です。

ここで、今回の規定課題では、解ベクトル x が1本である保証はありません。 すなわち、m 本の解ベクトルをまとめた行列 X を

X = (x1 x2 ... xm)

とし、m 本の右辺ベクトル b をまとめた行列 B を

B = (b1 b2 ... bm)

とすると、

A X = B

の解ベクトル行列 X を解く問題と定義できます。

予選課題詳細

具体的に問いていただくのは以下の問題です。なお，これらの問題は予め ツールキット(正式版)の lib.o 内に含まれていますので， 各自で実装する必要はありません。 解精度がツールキットの3倍を超えない範囲で高速化を図ってください。 詳細はツールキット(正式版)およびツールキット 同梱のドキュメントを ご覧ください。

なお，コンテストで使用する問題の入力データサイズは， 最大利用可能領域サイズの 1/2 以下であることを保証いたします。 また，コンテストではPPE プログラムの変更は禁止いたします。 変更されている場合は失格となりますので，ご注意ください。

コンテスト規定課題の進め方に関しては，「競技形式」 のページも併せてご覧ください。

問題1

        /* 密非対称乱数行列，N=*，M=** */
        for(j=0;j<n;j++){
            for(i=0;i<n;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }

問題2

        /* 密非対称乱数行列，N=*，M=1 */

        for(j=0;j<n;j++){
            for(i=0;i<n;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }

問題3

        /* 非対称三重対角行列，N=*，M=1 */
        for(i=0;i<2;i++){
            a[i] = ((float)rand()/(float)RAND_MAX);
        }
        for(j=1;j<n-1;j++){
            for(i=j-1;i<j+2;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }
        for(i=j-1;i<n;i++){
            a[n*(n-1)+i] = ((float)rand()/(float)RAND_MAX);
        }

問題4

        /* 非対称帯行列，N=*，M=** */
        d = n / 4 + 1;
        hd = (d-1) / 2;
        for(j=0;j<hd;j++){
            for(i=0;i<j+hd+1;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }
        for(j=hd;j<n-hd;j++){
            for(i=j-hd;i<j+hd+1;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }
        for(j=n-hd;j<n;j++){
            for(i=j-hd;i<n;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }

問題5

        /* 非対称対角優位行列，N=*，M=** */
        for(j=0;j<n;j++){
            for(i=0;i<n;i++){
                a[n*j+i] = ((float)rand()/(float)RAND_MAX);
            }
        }
        for(j=0;j<n;j++){
            a[n*j+j] = a[n*j+j] + (float)n;
        }
