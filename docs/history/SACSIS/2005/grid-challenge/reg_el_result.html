決勝スロット割り当て

  • 全体スケジュールは3/7(月)-3/18(金)のうちの平日、昼間(普通の人が働く時間)です。
  • 一チーム2:30のスロットが割り当てられます。
  • 予選を突破したチームは希望スロットを以下から申告して下さい (grid-challenge-secretary@logos.ic.i.u-tokyo.ac.jp)。 自分より上位のチームが未定の場合、必要なだけ第2, 第3希望などを書いておいて下さい (例: 自分の10位が確定しているが、時間割り当て表ではまだ4位までしか決まっていない => 申告時に第6希望まで書いておけば確実)。
  • 決勝では割り当てられたスロット時間内に本選問題を解き、以下の物を 提出して下さい。
    • 結果の要約("<< "と">> "でくくられた出力行)
    • 問題に2度連続して正解した模様を収めたビデオ (画面の動画キャプチャ)。以前に5回と書きましたが、2回に 変更します。また、一回は動いたが2回目で失敗したまま時間切れ、などの場 合も、できた範囲の物を提出して下さい。
  • 画面をキャプチャは、WindowsであればWindows メディアエンコーダ を用いて簡単にできます。Windowsから cygwinのssh や、Tera TermでGCF環境にログ インしたり、VMWareを用いてWindows内でLinuxを動かし、そこからGCF環境に ログインしたりして、なんとか撮影してくれることを望みます。Tera Termを使う場合、 TTSSHを合わせ てインストールして下さい。どうしても無理という 場合はご相談下さい。

決勝期間中のマシン利用について

決勝期間中は、以下の時間帯にGCFクラスタへのログインと練習が可能です。
  • 決勝期間中の、21:00から翌日の8:00まで (3/6 21:00--3/7 8:00を含む)
  • 3/12(土), 3/13(日)
  • どのチームにも割り当てられていない時間帯、およびそのような二つの時間帯の 間の30分間

具体的には以下の時間帯にログイン可能です。

3/6(日) 21:00 - 24:00
3/7(月) 0:00 - 24:00

以下の時間帯にはログインしないでください。

  • 他チームのスロット、およびその直前30分間(朝一のスロットについては1時間)
  • 予選終了時から3/6 21:00まで
なお、決勝用のAPI/データは自チームのスロット以外の期間には利用できません。 予選用のデータは上記のログイン可能とされている時間帯に利用可能です。
