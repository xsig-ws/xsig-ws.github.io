Grid Challenge in SACSIS2005

概要

SACSISの併設企画Grid Challenge in SACSIS 2005では，国内のGrid研究の拠 点の協力により, 1,000プロセッサ規模の計算機資源 を集め，それを企画開催中の一定期間，参加者に提供します ( Gangliaによるモニタリング情報)．

その環境で，与えられた問題を解く速度を競う「規定課 題部門」ならびに与えられた資源を用いて興味深い実験を実行，報告 してもらう「自由課題部門」を設けます．

規定課題部門では，成績優秀者への表彰を行い，登壇発表をしていただく 予定です．自由課題部門では，ある程度の規模での動作実績があるソフトウェ アを対象に，本環境での動作実験の報告を歓迎します．特に，大規模な広域環 境での問題---スケーラビリティ，容易な運用性，異機種性，通信バンド幅や 遅延，など---に焦点を当てた実験の報告を歓迎します．選ばれた報告に関し ては，SACSIS 2005のポスターセッションにて発表をしていただきます．さら にその中から選ばれた少数の報告に関しては，SACSIS 2005にて特別セッショ ンをもうけ，登壇発表をしていただきます．

規定課題部門の参加は学生に限りますが, 自由課題部門は，大規模な環境 を利用することに意味のある実験であれば，どなたの応募でも受け付けます．

部門

規定課題部門: Grid並列プログラミングコンテスト(PSC on Grid)
規定の問題を，与えられた計算機資源(またはその一部)を用いて高速に解 く．資源を全員に解放してその間にとにかく正解を出す「予選」と，予選 を通過したチームが順番に排他的に利用できる期間を設けて速度を記録する 「決勝」を設けます．決勝は短時間での勝負です．問題と詳細ルールは こちらです
自由課題部門: 大規模広域分散環境を体験しよう
多数の広域に分散した資源が与えられるまたとないチャンスを用いて， Grid コミュニティに関わる人々で結果を共有できる，興味深い実験を行って 下さい．SACSIS 2005において実験の報告を行うセッションをもうけ，性能報 告，(不)動作報告，ノウハウ共有，などをしていただきます． 自由課題の例とルールはこちらです

資源を提供する拠点

本企画で利用することができる計算機資源群を, Grid Challenge Federated Clusters, またはGrid Challenge Federation (GCF)と呼びま す.

以下は, GCFに計算機資源を提供している拠点・研究室です． それは同時に，本企画のために多大な 人的・計算機資源的・電気代的 協力をしてくださったグループです．計算機資源に関する詳細な情報は，参加 者にのみ提供されます．総計で1,000プロセッサにおよぶ資源を用いた並列処理 を是非体験してみて下さい.

  • 産業技術総合研究所 グリッド研究センター
  • 筑波大学 HPCS研究室
  • 電通大 弓場研究室・本多研究室
  • 東工大 松岡研究室 合田研究室
  • 徳島大 小野研究室
  • 東大 平木研究室
