Grid Challenge in SACSIS 2005

賞について

Grid Challengeの規定部門では以下の賞が提供されます。
予選 本選
初級 ○ ○
上級 ○ ○
  • 予選は、一番「早く」(early)正解を出したチームに賞が与えられます。 それと同時に、予選に早く正解を出した順に、本選の時間帯を選択します。
  • 本選は、一番速いプログラムを書いたチームに賞が与えられます。
  • それぞれについて「初級」のデータ( 規定問題 詳細のページ参照)のみを対象とする部門と、すべてのデータを対象とす る部門を設けます。もちろん、すべてのデータに正解したチームは初級部門で も、評価の対象となります。ただし、両方の部門で同時に最優秀となった場合、 賞は上級部門の方で与えられます。その場合、初級部門の賞は次点のチームに 与えられます。
  • 賞を得たチームは、SACSIS 2005にて表彰されます。また、副賞は現在交 渉中です。
